In this challenge, you are required to draw a small cabin out of unicode characters. You will be given two parameters width (odd number in range <5, 49 >) and height (number in range <5, 50 >). Based on these inputs you are required to draw a log cabin with a fence that will be width characters wide and height lines tall, the fence should always be 2 lines high with a supporting beam on each sides of the cabin. To make it a challenge, the goal is to build it as fast as possible.

Example 1

Input: height:10, width:11

Output:

x x x x x x x xxxxxxxxx x x x x x x =x=======x= |xxxxxxxxx|

Example 2

Input: height:10, width:7

Output:

x x x xxxxx x x x x x x x x x x =x===x= |xxxxx|

Example 3

Input: height:5, width:7

Output:

x x x xxxxx =x===x= |xxxxx|
